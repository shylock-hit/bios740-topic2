# BIOS 740 Topic 2 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a reproducible code package and report for BIOS 740 Topic 2: biomedical NER and relation extraction on ADKG and MDKG.

**Architecture:** The project will be a lightweight Python package with command-line scripts for downloading/loading data, EDA, SpERT-format conversion, metrics, error analysis, and cross-domain transfer experiment orchestration. Training hooks will target SpERT, while local tests cover deterministic data handling, conversion, and evaluation logic without requiring a GPU.

**Tech Stack:** Python 3.10+, pytest, pandas/matplotlib optional for EDA, JSONL/JSON artifacts, shell scripts for reproducible workflows.
